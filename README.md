Skeleton project of Next and Redux


## Prerequisite

- Node JS : v12.19.0
- NPM : 6.14.8

## Commands

### Instalation (First Time Only)

```
npm install
```

### Run on Development

```
npm run dev
```

Apps will run on port 20203

### Build and Run on Production

```
npm run build && npm run start
```
